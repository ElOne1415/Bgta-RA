# User Manual — Bogotá Verde AR

## Objective

Bogotá Verde AR is a mobile WebAR prototype that uses augmented reality to present information about Bogotá's public parks, environmental care, public space rules, safety recommendations and citizenship culture.

## Requirements

- Mobile phone with camera.
- Internet connection.
- Safari or Chrome browser.
- GitHub Pages HTTPS link.
- Official Hiro marker opened on another screen or printed.

## Steps

1. Open the project URL on the phone.
2. Tap **Start AR**.
3. Accept camera permission.
4. Point the camera at the Hiro marker.
5. Keep the marker flat, complete and well lit.
6. Use **Previous** and **Next** to change the AR content.
7. Open **Menu** only if you want to jump to a specific section.
8. Use **Mini Quiz** to validate learning.

## Presentation fallback

If the camera works but the marker is not detected during the presentation, press **Show demo without marker**. This displays the 3D educational content over the camera view so the prototype can still be explained.

## Troubleshooting

- If the camera does not open: use HTTPS and open the page directly in Safari or Chrome.
- If the camera opens but no 3D appears: the marker is not being detected.
- If detection is unstable: increase light, avoid reflections, keep the black border visible and move the phone 30–50 cm away.
- If the page was opened from WhatsApp: open it externally in Safari or Chrome.

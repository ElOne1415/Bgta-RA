# Bogotá Verde AR — User Manual

## 1. What this prototype does

Bogotá Verde AR is a marker-based Web Augmented Reality prototype. It uses the phone camera to detect a Hiro marker and display an interactive 3D learning object about Bogotá's public parks.

The prototype includes:

- A 3D public park model.
- Informational AR cards.
- Sections about parks, environmental care, public space rules and safety.
- A mini quiz for learning validation.
- A poster page with an automatic QR code.
- A printable Hiro marker page.

## 2. Required device

Use a smartphone with:

- Camera access.
- Internet connection.
- Modern browser: Chrome on Android or Safari/Chrome on iPhone.

Important: camera access on mobile requires HTTPS. For that reason, the project should be uploaded to GitHub Pages before testing it on a phone.

## 3. Files included

- `index.html`: main AR prototype.
- `app.js`: interactive menu, AR content and quiz logic.
- `styles.css`: visual design and responsive layout.
- `marker.html`: printable Hiro marker.
- `poster.html`: poster with QR code and marker preview.
- `README.md`: project overview.
- `deploy-github-pages.md`: GitHub Pages deployment steps.

## 4. How to use it

1. Upload the project to GitHub Pages.
2. Open `poster.html` and print or display the poster.
3. Open the QR code with a phone.
4. Press **Start AR Experience**.
5. Allow camera access.
6. Point the camera at the Hiro marker.
7. Use the menu to explore the AR content.
8. Complete the mini quiz.

## 5. Tracking recommendations

For better marker detection:

- Keep the marker flat.
- Use good lighting.
- Avoid reflections or shadows.
- Keep the marker fully visible.
- Keep the camera around 30–50 cm from the marker.
- Do not cover the black border.

## 6. Presentation script idea in English

This prototype is called Bogotá Verde AR. It is a marker-based augmented reality learning object designed to teach users about public parks in Bogotá. When the user scans the Hiro marker, a 3D park model appears with interactive information cards. The menu allows users to explore representative public spaces, environmental care recommendations, public space rules, safety tips and a mini quiz. The main purpose is to transform static educational information into a more visual, interactive and memorable learning experience.

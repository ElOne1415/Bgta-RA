# Bogotá Verde AR - WebAR Prototype

This is a simple marker-based WebAR prototype for the project **Bogotá Verde AR**.

## What it includes

- A WebAR scene activated with the standard Hiro marker.
- A simple 3D park with trees, benches, trash bin, signs, and floating labels.
- An interactive menu with these sections:
  - Home
  - Map
  - Simón Bolívar Park
  - National Park
  - Parque de los Novios
  - Environmental Care
  - Public Space Rules
  - Safety Tips
  - Mini Quiz

## How to test it locally

1. Open `marker.html` on a computer or print it.
2. Open `index.html` on a phone with camera access.
3. Allow camera permissions.
4. Point the camera at the Hiro marker.
5. Use the menu buttons to change the AR information.

## Important

For a real public presentation, host the folder on a static hosting service with HTTPS. Mobile browsers usually require HTTPS to access the camera.

After publishing the site, generate a QR code using the public URL of `index.html` and place it on the poster.

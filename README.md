# Bogotá Verde AR

Final WebAR prototype for the interface experimentation project.

## Project title

**Bogotá Verde AR: Interactive Augmented Reality Experience for Learning About Public Parks and Responsible Citizenship**

## Problem

Traditional educational material about public parks is usually static. Students and visitors may read information, but they do not always interact with it or remember it. This prototype uses augmented reality to make the learning process more visual, interactive and engaging.

## Objective

To design and implement a marker-based Web Augmented Reality prototype that presents educational content about Bogotá's public parks, environmental care, public space rules and safety recommendations.

## Type of AR

Marker-based Web Augmented Reality using AR.js and A-Frame.

## Contents created in AR

- 3D park model.
- Floating information cards.
- Representative park points.
- Environmental care recommendations.
- Public space rules.
- Safety tips.
- Mini quiz.

## How to run

Open `index.html` from a HTTPS server. For mobile testing, deploy it on GitHub Pages.

Local file opening is not recommended because mobile browsers require HTTPS for camera access.

## Main files

- `index.html` — AR application.
- `app.js` — Interactions and content.
- `styles.css` — Interface styles.
- `marker.html` — Hiro marker.
- `poster.html` — Poster and QR code.
- `USER_MANUAL.md` — User instructions.
- `deploy-github-pages.md` — Deployment steps.

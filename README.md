# Bogotá Verde AR - Stable Mobile Prototype

This version is designed to avoid the black-screen camera problem on iPhone/Android.

## How to use
1. Upload all files to the root of your GitHub Pages repository.
2. Open the public HTTPS URL directly in Safari or Chrome.
3. Press **Start camera**.
4. Open or print `marker.html` on another device/paper.
5. Place the marker inside the frame and press **Marker centered · Show AR**.
6. Use **Next / Previous / Menu** to explore the content.

## Important
Do not open the page inside WhatsApp, Instagram or Facebook internal browser.
Use Safari on iPhone or Chrome on Android.

This is a stable WebAR exhibition mode: it uses the real mobile camera and places an augmented 3D park scene over the camera view, guided by the Hiro marker.
